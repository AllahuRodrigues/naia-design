# Naia de Sousa - Creative Portfolio

A modern, interactive portfolio website for Naia de Sousa, showcasing her work as a Mozambican visual artist. Built with React, Three.js, and modern web technologies.

## 🚀 Features

- Interactive 3D experiences using Three.js
- Responsive design with Tailwind CSS
- Smooth animations with Framer Motion
- Modern UI with styled-components
- Contact form with email functionality
- Image gallery with lazy loading
- SEO optimized

## 🛠️ Tech Stack

- React 18
- TypeScript
- Three.js
- Framer Motion
- Tailwind CSS
- Styled Components
- Parcel Bundler
- Node.js

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/AllahuRodrigues/naia-design.git
cd naia-design
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🏗️ Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/         # Page components
├── styles/        # Global styles and theme
├── utils/         # Utility functions
├── assets/        # Static assets
└── server/        # Backend server code
```

## 🔧 Configuration

- `tailwind.config.js` - Tailwind CSS configuration
- `postcss.config.js` - PostCSS configuration
- `tsconfig.json` - TypeScript configuration
- `.eslintrc.js` - ESLint configuration
